Components of the World Wide Web

HTML
The foundational coding language used to structure content on the web, such as text, images, and links.
HTTP
The system used to transfer data between a web server and a browser, enabling the loading of web pages.
URL
A specific type of address that provides the location of a resource on the web, such as a webpage or file.
